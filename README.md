# Beastmaster
A MMORPG focused around capturing creatures and fighting with them in **real-time** battles.

Backend is written in python, client emulates server using javascript

## Overmap
 * Quests
 * Items
 * Shops
 * Player chat

## Battles
 * Abilities
 * Tile based movement
 * Projectiles are unrestricted
 * n vs n creatures
